## Installation

```bash
# Download
git clone my-site https://github.com/fedonman/fedonman-website.git

# Install dependencies
cd my-site
npm install

# To develop
npm run develop

# To build
npm run build

# To run compiled
npm run serve
```

*Some features may not worked when served locally instead of Gatsby cloud.*
